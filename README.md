# 💀 Smart Excuse AI

Smart Excuse AI is an AI-powered web application that generates believable academic excuses based on user situations.  
The system uses Natural Language Processing (NLP) and Machine Learning to classify situations, generate context-aware excuses, evaluate their believability, and provide AI explanations.

The goal of the project is to demonstrate how AI pipelines can be used to build interactive and intelligent web applications.

---

## 🌐 Live Demo

Try the app here:

https://smart-excuse-ai.streamlit.app/

---

## 🚀 Features

- Situation classification using machine learning
- Context-aware academic excuse generation
- AI rewrite engine to improve excuses
- Believability scoring using Logistic Regression
- Professor suspicion risk analysis
- Explainable AI reasoning
- Personality modes (Serious / Balanced / Chaotic)
- User feedback learning system
- Automatic model retraining
- Excuse history tracking
- Difficulty meter for excuse effectiveness
- Web interface built with Streamlit

## 🧠 AI Pipeline

The system follows a multi-stage AI pipeline:

User Input
↓
Situation Classifier (NLP + ML)
↓
Excuse Generator
↓
Rewrite Engine
↓
Realism Detector
↓
Excuse Ranking
↓
Confidence Analyzer
↓
Explanation Engine
↓
Final Excuse Output
